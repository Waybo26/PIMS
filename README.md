# Patient Information Management System

## Team

## Description
A centralized patient information management system to streamline record keeping, administrative reporting, and financial operations.

## Requirements
- Python 3.8+
- pandas
- matplotlib
- pytest

## Execution
Run the main script:
`python main.py`

Run the tests:
`pytest test_system.py`
